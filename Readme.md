
## Requirements
* [Python](https://www.python.org) v3.5 or greater

## Installation

1. Clone repository
```
$ git clone https://github.com/stsupertam/Ku-Web-Crawler
```

2. Install python dependencies
```
$ cd Ku-Web-Crawler
$ pip3 install -r requirements.py
```

## How to run
```
$ cd Ku-Web-Crawler
$ python main.py
```
